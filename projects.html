ORC (Graphics & Simulation Engine)

A custom graphics rendering pipeline migrated from a 2D raycaster to a complete 3D BSP-driven pipeline using WebGL2. Features true Z-coordinates, billboarded sprites, and custom convex/concave sector drawing tools.

Asteroids Game Using ECS

A simple Asteroids game built with vanilla JavaScript and HTML5 Canvas, this is an implementation of the Entity-Component-System (ECS) architecture. It features a modular design, allowing for easy addition of new entities, components, and systems. The game includes player-controlled spaceship, asteroids, and shooting mechanics.

Map Editor For ORC Engine

A map editor built with vanilla JavaScript and HTML5 Canvas for the ORC Engine. Allows users to create and edit 2D and 3D maps with support for convex and concave sectors, texture mapping, and sprite placement. and uses half edge data structures to find faces of closed shapes.
